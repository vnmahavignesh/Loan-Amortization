# Loan Amortization Calculator

A comprehensive loan amortization calculator with prepayment tracking and detailed reporting features.

## Features

- **Loan Calculation**: Calculate EMI based on loan amount, interest rate, and tenure
- **Amortization Table**: Detailed month-by-month breakdown of payments
- **Prepayment Tracking**: Track multiple prepayments per month with dates and descriptions
- **Other Charges**: Record additional charges with detailed information
- **Year-wise Filtering**: View data by specific years
- **Export Functionality**: Export data to CSV format
- **Auto-save**: Automatic saving of data to browser storage
- **Responsive Design**: Works on desktop and mobile devices
